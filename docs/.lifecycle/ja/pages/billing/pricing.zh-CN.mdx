# 価格

Zeaburは、従量制の価格設定モデルを提供します。リソースを使用しない場合でも請求書は届きません。

代わりに、Zeaburはサービスの実際の利用に基づいて、1分ごとのvCPUとメモリリソースの料金を請求します。

## サービス料金

Zeaburは以下の式に従って利用料金を計算します：

`合計サービス時間`（分単位）×（`メモリ使用量`（GB単位）+ `vCPU使用量`（コア単位））

そして、利用料金は次のとおりです：

- メモリ - **US$0.00023** / GB-分（約 US$10 / GB-月）

- vCPU - **US$0.0005** / vCPU-分（約 US$20 / vCPU-月）

- アウトバウンドトラフィック - Serverlessプランのユーザーには、最初の10GBのトラフィックが無料です。DeveloperプランとTeamプランの場合、100GBまで無料です。超過分の料金は **US$10** / 100GB です。

サービス料金には最低料金はありません。サービスがリソースを使用しない場合やクラッシュした場合も、料金は請求されません。
