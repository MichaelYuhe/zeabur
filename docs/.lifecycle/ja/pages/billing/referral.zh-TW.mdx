# レコメンデーション

Zeaburは、すべてのユーザーにレコメンデーションプログラムを提供しています。友達をZeaburに招待してポイントを獲得しましょう。

あなたの友達があなたのレコメンデーションリンクを使って登録し、有料プランにアップグレードすると、あなたのアカウントに5ドル分のポイントが追加されます。

## 友達を招待する方法

まず、友達を招待するにはあなたのレコメンデーションリンクが必要です。[レコメンデーション](https://dash.zeabur.com/account/referral) ページで自分のレコメンデーションリンクを見つけることができます。

あなたのレコメンデーションリンクをコピーし、そのリンクを友達と共有してください。メール、ソーシャルメディア、または他にあなたが好むあらゆる方法で共有できます。

## ポイントの確認方法

あなたの獲得したポイントは [レコメンデーション](https://dash.zeabur.com/account/referral) ページで確認できます。

![レコメンデーション](/referral/referral.png)

ご覧の通り、このページには２種類のレコメンデーション状態が表示されます。一つは保留中で、これはあなたの友達があなたのレコメンデーションリンクを使って登録はしたが、まだ有料プランにアップグレードしていないことを意味します。もう一つは完了で、これはあなたの友達が有料プランにアップグレードし、あなたがポイントを受け取ったことを意味します。

## ポイントの使用方法

ポイントは支払い時に最優先で使用されます。使用量料金とサブスクリプション料金はまずポイントから差し引かれ、ポイントが消耗した後に登録された支払い方法から料金が請求されます。
