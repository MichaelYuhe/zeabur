# 料金

Zeaburは、プロジェクトに向けた消費ベースの料金体系を提供しています。使用しないリソースに対する請求はありません。

代わりに、Zeaburは、サービスが実際に使用する各vCPUおよびメモリリソースに対して分単位で料金を請求します。

## 料金設定

Zeaburは、以下の数式で使用料金を計算します:

`総サービス時間` (分単位) x (`メモリ使用量` (GB単位) + `vCPU使用量` (コア数))

そして、使用料金は以下の通りです:

- メモリ - **US$0.00023** / GB-minute (約US$10 / GB-month)

- vCPU - **US$0.0005** / vCPU-minute (約US$20 / vCPU-month)

- ネットワークの転送 - サーバーレスプランでは最初の10GBは無料、開発者プランやチームプランでは100GBまで無料。その後、**US$10** / 100GB。

最小サービス費用はありませんし、サービスがリソースを使用しない場合やクラッシュした場合にも請求されることはありません。
