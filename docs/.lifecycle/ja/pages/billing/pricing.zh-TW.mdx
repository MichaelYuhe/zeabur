# 価格

Zeabur はプロジェクトに利用量に基づく価格モデルを提供し、使用していないリソースに対して請求されることはありません。

逆に、Zeabur は実際に使用されたサービスの vCPU とメモリリソース使用量に応じて料金を請求します。

## サービスの料金

Zeabur は以下の公式に基づいて料金を計算します：

`総サービス時間`(分) x (`メモリ使用量`(GB) + `vCPU 使用量`(コア))

そして、リソース使用に対する毎分の料金は以下の通りです：

- メモリ - **US$0.00023** / GB-分 (約 US$10 / GB-月)

- vCPU - **US$0.0005** / vCPU-分 (約 US$20 / vCPU-月)

- アウトバウンドトラフィック - Serverless Plan のユーザーには最初の 10GB は無料で、Developer Plan と Team Plan は 100GB 無料です。それを超えた分の料金は **US$10** / 100GB です。

サービスの最低料金はなく、使用されていないリソースやクラッシュした場合でも料金は発生しません。
