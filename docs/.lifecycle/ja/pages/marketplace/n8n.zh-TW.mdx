# n8n のデプロイ

[n8n](https://github.com/n8n-io/n8n) は拡張可能なワークフローオートメーションツールです。フェアコード配布モデルを採用していて、n8n は常にソースコードが閲覧可能で、自己ホスティングが可能、独自のカスタム機能、ロジック、アプリケーションの追加を許可します。n8n のノードベースのアプローチは非常に柔軟で、あらゆるものを何にでも接続することができます。

この記事では、PostgreSQL を使用して Zeabur 上で n8n をデプロイする方法を説明します。

## ステップ1：プロジェクトの作成

まず、Zeabur 上にプロジェクトを作成する必要があります。[プロジェクト](https://dash.zeabur.com/projects) ページに移動すると、すべてのプロジェクトが表示され、「プロジェクトを作成する」ボタンがすべてのプロジェクトの最後にあります。

**プロジェクトを作成する**ボタンをクリックし、プロジェクト名を入力すると、新しいプロジェクトが作成されます。

## ステップ2：PostgreSQL サービスのデプロイ

n8n はユーザーの識別情報を PostgreSQL データベースに保存します。「サービスを作成する」をクリックし、「サービスマーケット」を選択すると、PostgreSQL がリストに表示されます。

![PostgreSQL のデプロイ](/marketplace/postgresql/deploy.png)

**PostgreSQL** サービスを選択すると、Zeabur が自動的にそのサービスのデプロイを開始します。

## ステップ3：データベースの作成

n8n サービスを正常に稼動させるためには、`n8n` という名前の PostgreSQL データベースを先に作成する必要があります。

通常は、PostgreSQL クライアントが必要で、PostgreSQL サービスに接続しデータベースを作成します。しかし、Zeabur はデータベースを作成するための便利な方法を提供しています。

PostgreSQL サービスの中に **Database（データベース）** というタブがあり、そこでクリック操作だけでデータベース管理ができます。

![データベースタブ](/marketplace/n8n/database-tab.png)

**n8n** は `n8n` という名前のデータベースが必要なので、`n8n` という名前のデータベースを作成する必要があります。
入力ボックスに `n8n` を入力し、**作成**ボタンをクリックすると、Zeabur が `n8n` というデータベースを作成します。

![データベースタブ](/marketplace/n8n/database-n8n.png)

## ステップ4：n8n サービスのデプロイ

PostgreSQL サービスを正常に作成し、データベースを設定した後、n8n のデプロイを開始できます。
**PostgreSQL** サービスを作成したばかりのプロジェクトページに戻り、「サービスを作成する」をクリックし、「サービスマーケット」を選択すると、リストで n8n を見つけることができます。

![n8n のデプロイ](/marketplace/n8n/deploy.png)

**n8n** サービスを選択すると、Zeabur が自動的にそのサービスのデプロイを開始します。

<Callout>
  PostgreSQL をデプロイする前に n8n をデプロイした場合、PostgreSQL
  サービスをデプロイした後に n8n サービスを再起動する必要があります。これにより、n8n は正しいデータベース接続情報を取得できます。
</Callout>

## ステップ5：ドメインのバインド

Zeabur での n8n サービスの「ドメイン」セクションで、ドメインをバインドできます。

このドメインは、自分のドメインであるか、または Zeabur が提供する無料のドメインであることができます。

サービスにドメインをバインドする方法については、[ドメインのバインド](../deploy/domain-binding)を参照してください。

## ステップ6：アカウントの設定

デプロイが完了したら、バインドしたドメインから n8n サービスにアクセスできます。以下のようなページが表示されます：

![n8n セットアップ](/marketplace/n8n/setup.png)

ページの指示に従って n8n アカウントを設定します。設定が完了すると、n8n を正常に使用できるようになります。

![完了](/marketplace/n8n/finish.png)
