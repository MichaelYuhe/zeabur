# RSSHubサービスのデプロイ

[RSSHub](https://rsshub.app) はオープンソースで使いやすく、拡張可能なRSSフィード生成器で、変わったコンテンツにもRSSフィードを生成できます。RSSHubはオープンソースコミュニティの力を借りて素早く発展しており、現在は数百のウェブサイトの数千に対応しています。

![RSSHub](/marketplace/rsshub/intro.png)

この記事では、Zeaburのサービスマーケットを通じて自分のRSSHubサービスをデプロイし、自分のドメインをバインドする方法について説明します。

## ステップ1：プロジェクトの作成

第一歩としてプロジェクトを作成する必要があります。[プロジェクト](https://dash.zeabur.com/projects)ページに進むと、プロジェクトリストが表示されます。

リストの最後にある"プロジェクトの作成"ボタンをクリックして新しいプロジェクトを作成します。

![Create Project](/deploy/common/create-project.en-US.png)

## ステップ2：RSSHubサービスのデプロイ

サービスマーケットでRSSHubサービスをワンクリックでデプロイできます。

"cmd + k"を押すか、"サービスを追加"ボタンをクリックしてから"サービスマーケット"を選択します。

![Select Marketplace](/deploy/common/select-service.en-US.png)

リストからRSSHubを見つけてクリックすると、デプロイが始まります。

## ステップ3：ドメインのバインド

Zeaburのコントロールパネルで、RSSHubサービスの"ドメイン"セクションでドメインをバインドできます。

このドメインは自分のドメインでも、Zeaburが提供する無料ドメインでも構いません。

サービスにドメインをバインドする方法については、[ドメインバインディング](../deploy/domain-binding)を参照してください。

## ステップ4：使用を開始する

デプロイが完了したら、バインドされたドメインでRSSHubサービスにアクセスできます。

好きなRSSフィードを選んでRSSリーダーで登録しましょう。
