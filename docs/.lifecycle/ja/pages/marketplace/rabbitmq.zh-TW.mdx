# RabbitMQ サービスのデプロイ

Zeabur は RabbitMQ サービスを提供しており、Zeabur 上でワンクリックで RabbitMQ メッセージキューをデプロイできます。

<Callout type="info" emoji="ℹ️">
    現在の RabbitMQ サービスのバージョンは 3-management です。
</Callout>

## RabbitMQ サービスのデプロイ

あなたのプロジェクトで、**サービスの作成** → **Prebuilt** をクリックし、ポップアップされるリストの検索ボックスで `RabbitMQ` を検索します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイをクリックすると、Zeabur は自動的に RabbitMQ サービスをデプロイしてくれます。

## 環境変数

RabbitMQ サービスをデプロイすると、Zeabur は関連する環境変数を自動的に他のサービスに注入します。

 - `RABBITMQ_HOST`
 - `RABBITMQ_PORT`
 - `RABBITMQ_DEFAULT_USER`
 - `RABBITMQ_DEFAULT_PASS`

場合によっては、自分で追加した `DATABASE_URL` を使用して、上記の環境変数を置き換えることができます。例えば：

```bash
amqp://<RABBITMQ_DEFAULT_USER>:<RABBITMQ_DEFAULT_PASS>@<RABBITMQ_HOST>:<RABBITMQ_PORT>/<DATABASE_NAME>
```

ここでの `<DATABASE_NAME>` は、あなたが自分で追加した Vhost 名前です。

## オンライン管理インターフェース

RabbitMQ サービスはオンライン管理インターフェースを提供しており、ブラウザで RabbitMQ サービスを管理できます。

`http://<RABBITMQ_HOST>:15672` にアクセスして、RabbitMQ サービスのオンライン管理インターフェースを使用できます。

## DSN 設定

`${RABBITMQ_URI}` の変数を通じて、RabbitMQ サービスの DSN（つまり、RabbitMQ サービスのネットワークアドレス）を使用することができます。
