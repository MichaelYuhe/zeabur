# 導入

**Zeabur**は、クリックひとつでサービスをデプロイできるプラットフォームです。どのプログラミング言語やフレームワークを使用しているかにかかわらず、利用できます。


詳細を学び、優先サポートを受けるために、[Discordサーバー](https://discord.gg/DrdGCvXEyY) に参加できます！

### Zeaburの利点

#### 便利で迅速

従来のデプロイメント手法と比較して、Zeaburを使用すると、多くの時間をデプロイに関する知識の学習に費やさず、たった数分でデプロイを完了することができます。これにより、時間とエネルギーを製品開発に注力できます。

- 即戦力のCI/CD
- プロジェクトタイプの自動認識
- 迅速なドメイン生成
- 複数環境の分離

#### 利用量に応じた支払い

Zeaburは、利用した実際のリソースのみを支払う従量制の料金体系を採用しています。つまり、サービスで使用した実際のリソースだけを支払う必要があり、サーバー全体の料金を支払う必要はありません。

[料金に関するドキュメント](/en-US/billing/pricing) で詳細情報を確認できます。

#### オールインワンインテグレーション

Zeaburは、強力なオールインワンインテグレーション機能を提供しています。

フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージなど、さまざまなサービスを1つのプロジェクトにデプロイでき、それぞれを別々のプラットフォームに個別にデプロイする必要はありません。
