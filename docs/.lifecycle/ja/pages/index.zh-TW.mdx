# 基本紹介

**Zeabur** は、サービスをデプロイするのに役立つプラットフォームであり、ご使用のプログラミング言語やフレームワークに関係なく、わずか数回のクリックで完了できます。

詳細を知ったり優先サポートを受けたりするには、[Discord サーバー](https://discord.gg/DrdGCvXEyY) に参加することもできます！

### Zeabur の利点

#### 便利なデプロイ

従来のデプロイ方法と比較すると、Zeabur を使用すれば数分でデプロイでき、デプロイに関する知識を学ぶために多くの時間を費やす必要がありません。その代わりに、製品開発に時間と労力を注ぐことができます。

- 即座に使用可能な CI/CD
- プロジェクトタイプの自動識別
- 迅速なドメイン生成
- 複数環境の隔離

#### 従量課金

Zeabur は従量課金方式を採用しており、必要なリソースのみを支払う必要があります。つまり、サーバ全体の料金を支払う必要はありません。

詳細については[料金ページ](/billing/pricing)をご覧いただけます。

#### オールインワンの統合

Zeabur にはパワフルなオールインワンの統合機能が備わっています。

フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージなど、様々なサービスを一つのプロジェクトに同時にデプロイできます。それぞれを別々のプラットフォームにデプロイする必要はありません。
