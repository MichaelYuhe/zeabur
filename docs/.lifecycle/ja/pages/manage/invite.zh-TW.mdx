# メンバーを招待する

通常、1つのプロジェクトには複数のメンバーが関わります。プロジェクトのオーナーは他のメンバーを自分のプロジェクトに招待することができます。

### ステップ1：招待するメンバーのメールアドレスを入力する

プロジェクトの左上にある招待ボタンをクリックし、招待するメンバーのメールアドレスを入力し、招待ボタンをクリックします。

![email](/manage/invite/email.zh-TW.png)

### ステップ2：リンクを共有する（オプション）

招待ボタンをクリックすると、チームのメンバーは招待メールを受信し、メール内の同意をクリックすることでプロジェクトに参加できます。

また、招待後に生成される専用招待リンクを共有することもできます。このリンクを対応する招待先と直接共有することで、彼らはリンクをクリックすることであなたのプロジェクトに参加できます。

![email](/manage/invite/success.zh-TW.png)
