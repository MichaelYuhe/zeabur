# サービス利用量

Zeaburは従量制のモデルに従い、サービス利用量を詳細なグラフで提供しています。

表示したいサービスをクリックし、 **Metrics** ページを選択すると、実際の詳細な利用量グラフが表示されます。

### メモリ使用量

サービスの実行に使用されるメモリを表示します

![memory](/manage/metric/memory.png)

### CPU 使用量

サービスの実行で呼び出されるCPUを表示します

![memory](/manage/metric/CPU.png)
