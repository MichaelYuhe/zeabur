# メンバーを招待する

プロジェクトには通常、複数のメンバーが参加します。プロジェクトオーナーはメンバーを自分のプロジェクトに招待することができます。

### 第一ステップ：招待対象のメールアドレスを入力する

プロジェクト左上の招待ボタンをクリックし、招待対象のメールアドレスを入力し、招待ボタンをクリックします。

![email](/manage/invite/email.zh-CN.png)

### 第二ステップ：リンクを共有する（オプション）

招待ボタンを押した後、チームメンバーは招待メールを受け取り、メール内の同意するをクリックすることでプロジェクトに参加できます。

また、招待後に生成される専用の招待リンクを共有して、該当する招待対象にリンクを配布することもできます。リンクをクリックすることで、あなたのプロジェクトに入れます。

![email](/manage/invite/success.zh-CN.png)
