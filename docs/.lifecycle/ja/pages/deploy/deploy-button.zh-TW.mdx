# ワンクリックデプロイ

当社のワンクリックデプロイボタンを使用すると、ボタンを1クリックするだけでZeaburにサービスをデプロイできます。このボタンは、MarkdownファイルやHTMLファイルに使用することができます。[Zeaburのテンプレート](https://zeabur.com/templates)を自動的にZeaburにデプロイするプロジェクトに変換できます。

![button-demo](/deploy/button/button-demo.png)

## ボタンの機能

デプロイボタンは単純なHTMLコードの断片です。このコード断片をコピーして、任意のMarkdownファイルやHTMLファイルに貼り付けることができます。各ボタンは、利用者を所望のテンプレートにリダイレクトします。テンプレートは、単一のサービスまたはテンプレートの製作者によって決定されるサービスの組み合わせである可能性があります。

また、ボタンは製作者の推薦コードで終わることができ、推薦コードを使用して新規登録し、有料プランにアップグレードする各ユーザーに対して、製作者は5ドルのクレジットを獲得します。

## ボタンのコピー

各ボタンはテンプレートに関連しており、テンプレートの製作者だけがボタンを生成できます。

ボタンをコピーするには、[Zeabur Dashboard](https://dash.zeabur.com/account)のアカウントページで「テンプレート」オプションを選択し、作成したテンプレートのリストを参照する必要があります。

![created-templates](/deploy/button/created-templates.png)

必要なテンプレートを選んで共有ボタンをクリックすると、ボタンコードが含まれたポップアップが表示されます。

![popup](/deploy/button/popup.png)

推薦コードを有効にする場合は、ボタンコードに推薦コードが追加されます。推薦コードは、Zeaburでのユーザー名です。

## 使用シナリオ

これは、プロジェクトを世界に共有する最良の方法です。誰でも簡単にあなたのプロジェクトをワンクリックでデプロイできます。

例えば、以下にボタンを貼り付けることができます：

- GiHubのREADMEファイル
- 個人のウェブサイト
- ドキュメントとチュートリアル
- ソーシャルメディアのプロフィール

## 前提条件

デプロイボタンを取得するには、先にテンプレートを作成する必要があります。[Zeabur Dashboard](https://dash.zeabur.com/create-template)でテンプレートを作成できます。

下図のように、テンプレート名、説明を入力し、テンプレートに含めるサービスを選択するなど、フォームに必要事項を記入してテンプレートを作成します。

![create-template](/deploy/button/create-template.png)

## 報酬を獲得

本文書の冒頭で述べたように、あなたの推薦コードで新規登録し、有料プランにアップグレードする各ユーザーに対して、あなたは5ドルのクレジットを獲得します。クレジットを使用して自身のサービスの料金を支払うことができます。

詳細は、[推薦プログラムの文書](../billing/referral)で読むことができます。
