# カスタムルートディレクトリ

あるGitリポジトリ、例えばフロントエンドとバックエンドが同じにあるプロジェクトの場合、プロジェクトのルートディレクトリを手動で指定する必要があります。

### 第一歩：デプロイサービス

あなたのアプリケーションをデプロイするために[デプロイサービス](/zh-TW/get-started)を参照してください

### 第二歩：プロジェクトのルートディレクトリを変更する

デプロイされたサービスをクリックし、設定ページに入ります。スクロールダウンして**Root Directory**の項目まで
