# カスタムルートディレクトリ

一体型のフロントエンドとバックエンドアプリケーションなど、いくつかのリポジトリではプロジェクトのルートディレクトリを手動で指定する必要があるかもしれません。

### 第一歩：サービスのデプロイ

[サービスのデプロイ](/zh-CN/get-started)に従って、アプリケーションをデプロイしてください。

### 第二歩：プロジェクトルートディレクトリの変更

デプロイされたサービスを選択し、設定タブを選択します。 **Root Directory** セクションまで下にスクロールします。

![setting](/deploy/root-directory/setting.png)

指定したいルートディレクトリを入力し、保存をクリックします。保存に成功したら、**Redeploy Service** をクリックしてサービスを再デプロイし、サービスは指定されたルートディレクトリに基づいて再構築され実行されます。
