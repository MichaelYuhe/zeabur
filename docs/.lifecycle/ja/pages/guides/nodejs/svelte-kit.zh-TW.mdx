# Svelte Kit プロジェクトのデプロイ

Zeabur であなたの Svelte Kit プロジェクトをデプロイします。

### Step 1: Svelte Kit プロジェクトの作成（オプション）

ターミナルを開いて、次のコマンドを入力してください

```bash
npm create svelte@latest
```

次に、パッケージをインストールする必要があります：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install
    ```
  </Tab>
  <Tab>
    ```bash
    npm install
    ```
  </Tab>
  <Tab>
    ```bash
    yarn
    ```
  </Tab>
</Tabs>

プロジェクトが作成されたら、Git リポジトリを初期化して GitHub にデプロイしてください。

### Step 2: `@sveltejs/adapter-node` の設定

次に、`@sveltejs/adapter-node` をインストールする必要があります：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D @sveltejs/adapter-node
    ```
  </Tab>
</Tabs>

次に、`svelte.config.js` を開き、以下のように変更してください：

```js
import adapter from '@sveltejs/adapter-node'; // <-- ここに注意

export default {
  kit: {
    adapter: adapter({ out: 'build' })
  }
}
```

### Step 3: `package.json` の設定

`package.json` を開き、`scripts` に `start` コマンドを追加してください：

```json
{
  "scripts": {
    "start": "node ./build/index.js"
  }
}
```

### Step 4: Svelte Kit プロジェクトのデプロイ

プロジェクト内で **Deploy service** ボタンまたは **Add new service** ボタンをクリックし、 **Deploy your source code** を選択してください。

![deploy](/deploy/common/select-service.ja-JP.png)

デプロイしたい Svelte Kit コードリポジトリを検索し、インポートをクリックすると、Svelte Kit アプリケーションが自動的にデプロイされます。
