# Vite をデプロイする

Zeabur であなたの Vite アプリケーションをデプロイ。

### 第一歩：Vite アプリケーションの作成（オプション）

デバイス要件：
- Vite は Node.js バージョン 14.18+、16+ を必要とします。しかし、いくつかのテンプレートはより高い Node.js バージョンに依存して正常に動作するため、パッケージ管理システムから警告が出た場合は、Node.js のバージョンを更新することをお勧めします。

端末を開き、以下を入力します。
```bash
npx create vite@latest
# または
yarn create vite
# または
pnpm create vite
```

作成した後、Git リポジトリを初期化し、GitHub にデプロイします。

### 第二歩：Vite アプリケーションをデプロイする

プロジェクト内で、**Deploy service** または **Add new service** ボタンをクリックし、**Deploy your source code** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたい Vite アプリケーションのリポジトリを検索し、インポートをクリックすると、あなたの Vite アプリケーションが自動的にデプロイが始まります。
