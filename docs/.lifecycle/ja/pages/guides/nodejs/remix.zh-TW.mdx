# 部署 Remix プロジェクト

Zeabur であなたのRemixプロジェクトをデプロイします。

### 第1ステップ: Remixプロジェクトの作成（オプション）

環境要件：

- nodeのバージョンが14以上であることを確認してください

ターミナルを開いて、次のコマンドを入力します

```bash
npx create-remix@latest
```

作成が完了したら、Gitリポジトリを初期化し、GitHubにデプロイしてください。

### 第2ステップ: Remixプロジェクトのデプロイ

プロジェクト内で **Deploy service** または **Add new service** ボタンをクリックし、**Deploy your source code** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたいRemixプロジェクトのコードリポジトリを検索し、`import` をクリックすると、Remixプロジェクトが自動的にデプロイされます。
