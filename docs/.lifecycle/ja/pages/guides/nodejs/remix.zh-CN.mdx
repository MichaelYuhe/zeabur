# Remixのデプロイ

Zeabur上でRemixアプリケーションをデプロイします。

### 第一歩：Remixアプリケーションの作成（オプション）

デバイス要件：
- nodeのバージョンが14以上であることを確認してください

ターミナルを開いて、次のコマンドを入力します
```bash
npx create-remix@latest
```

作成が完了したら、Gitリポジトリを初期化してGitHubにデプロイします。

### 第二歩：Remixアプリケーションのデプロイ

プロジェクト内で、**Deploy service** ボタンまたは **Add new service** ボタンをクリックし、**Deploy your source code**を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたいRemixアプリケーションのリポジトリを検索してインポートをクリックすると、Remixアプリケーションのデプロイが自動的に始まります。
