# Next.js プロジェクトのデプロイ

この記事では、Zeabur で [Next.js](https://nextjs.org/) プロジェクトをデプロイする方法を学びます。

### 第一歩：Next.js アプリケーションの初期化（オプション）

ターミナルを開き、以下のコマンドを入力します：

```bash
npx create-next-app@latest
# または
yarn create next-app
# または
pnpm create next-app
```

このステップで問題が発生した場合は、[Next.js 公式ドキュメンテーション](https://nextjs.org/docs/getting-started)を参照してください。

作成が完了したら、Git リポジトリを初期化し、GitHub にプッシュします。

### 第二歩：Next.js プロジェクトのデプロイ

プロジェクト内で、**Deploy service** または **Add new service** ボタンをクリックし、**Deploy your source code** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたい Next.js プロジェクトのリポジトリを検索し、`import` をクリックすると、Next.js アプリケーションの自動デプロイが始まります。
