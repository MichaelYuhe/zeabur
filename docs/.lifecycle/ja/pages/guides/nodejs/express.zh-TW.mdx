# Express アプリケーションのデプロイ

[Express](https://expressjs.com/) は、軽量で柔軟な Node.js Web アプリケーションフレームワークであり、Web およびモバイルアプリケーションに強力な機能を提供します。

Zeabur 上で簡単に Express アプリケーションをデプロイできます。

### Step 1: Express アプリケーションの作成（オプション）

- コンピューターに Node.js がインストールされていること
- express-generator を使用するか手動で作成すること

```bash
# express-generator を使用
npx express-generator
```

詳細な手順については、Express の[公式ドキュメント](https://expressjs.com/en/starter/installing.html)を参照してください。

作成した後、Git リポジトリを初期化し、GitHub にデプロイします。

### Step 2: デプロイ

プロジェクトで、**新しいサービスを追加** ボタンをクリックし、**ソースコードをデプロイ** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたい Express アプリケーションリポジトリを検索し、インポートをクリックすると、Express アプリケーションが自動的にデプロイされます。
