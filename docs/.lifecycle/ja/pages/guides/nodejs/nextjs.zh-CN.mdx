# Next.js のデプロイ

Zeabur（ゼアバー）での Next.js アプリのデプロイ。

### 第一歩：Next.js アプリを作成する（オプション）

要件：
- node のバージョンが14以上であることを確認してください

ターミナルを開いて、以下を入力します。
```bash
npx create-next-app@latest
# または
yarn create next-app
# または
pnpm create next-app
```

作成が完了したら、そのプロジェクトを Git リポジトリとして初期化し、GitHub にデプロイします。

### 第二歩：Next.js アプリをデプロイする

プロジェクト内で、**Deploy service** または **Add new service** ボタンをクリックし、**Deploy your source code** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたい Next.js アプリのリポジトリを検索し、インポートをクリックすれば、Next.js アプリの自動デプロイが始まります。
