# Express アプリのデプロイ

[Express](https://expressjs.com/) は軽量で柔軟な Node.js ウェブアプリケーションフレームワークであり、ウェブおよびモバイルアプリへの強力な機能セットを提供しています。

Zeabur であなたの Express アプリをワンクリックでデプロイしましょう。

### 第一歩：Express アプリの作成（オプション）

- コンピュータに Node.js をインストール済みであること
- express-generator を使用するか、手動で作成する

```bash
# express-generator を使用する場合
npx express-generator
```

より詳細なチュートリアルは Express の[公式ドキュメント](https://expressjs.com/en/starter/installing.html)で見ることができます。

作成後、Git リポジトリを初期化して GitHub にデプロイします。

### 第二歩：デプロイ

プロジェクト内で、**Add new service** ボタンをクリックし、**Deploy your source code** を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたい Express アプリのリポジトリを検索してインポートすると、Express アプリのデプロイが自動的に開始されます。
