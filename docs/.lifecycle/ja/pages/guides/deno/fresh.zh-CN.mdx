# Freshアプリケーションのデプロイ

[Fresh](https://fresh.deno.dev/) は、JavaScriptおよびTypeScriptに適した現代的なフルスタックWebフレームワークです。

この記事では、ZeaburでFreshアプリケーションをワンクリックでデプロイする方法を説明します。

## Freshアプリケーションの作成（オプション）

- まずパソコンにDenoをインストールしてください。

- 詳しい手順は[Deno公式ドキュメント](https://deno.land/manual/getting_started/installation)でご覧になれます。

以下のコマンドを使用してDenoプロジェクトを初期化し、Freshフレームワークを使用します。

```bash
deno run -A -r https://fresh.deno.dev my-project
```

`main.ts`の中で、`port: Deno.env.get("PORT")`を`start`関数に追加して、Zeaburが提供する環境変数PORTでアプリケーションが適切にリッスンするようにします。

```typescript
// main.ts 内
await start(manifest, { plugins: [twindPlugin(twindConfig)], port: Deno.env.get("PORT")}, );
```

## アプリケーションの開発

`deno task start`を実行して開発サーバを起動します。

```bash
deno task start
```

次に、あなたのFreshプロジェクトの開発に取りかかるためにコードを変更していきます。

準備が整ったら、GitHubリポジトリを初期化してGitHubにアップロードしてください。


## アプリケーションのデプロイ

プロジェクト内で、**新サービスをデプロイ**ボタンをクリックし、**ソースコードをデプロイ**を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたいFreshプログラムのコードリポジトリを検索し、インポートをクリックすると、Freshアプリケーションの自動デプロイが開始されます。

デプロイが完了したら、 [ドメインのバインディング](../../deploy/domain-binding)のドキュメントを参考にして、外部からアクセス可能な公開ドメイン名をFreshプロジェクトにバインドする方法について確認することができます。
