# 基本的な紹介

**Zeabur**は、あなたがどのプログラミング言語や開発フレームワークを使用していても、いくつかの簡単なボタンを通じてサービスをデプロイするのを助けるプラットフォームです。

さらに詳しく知るため、そして優先サポートを得るためにも、私たちの[Discordサーバー](https://discord.gg/DrdGCvXEyY)に加入することができます！

### Zeaburの長所

#### 使いやすさと迅速さ

従来のデプロイ方法と比べて、Zeaburを使用すると、デプロイに関する知識を多くの時間をかけて学習することなく、数分でデプロイを完了させることができます。これにより、製品開発に時間とエネルギーを集中させることができます。

- オープンボックスのCI/CD
- プロジェクトタイプの自動識別
- 迅速なドメイン生成
- 複数環境の隔離

#### 使用量に基づく課金

Zeaburは使用量に基づく課金方法を採用しており、全サーバーの費用を支払うのではなく、サービスで実際に使用されたリソースにのみ費用を支払います。

より詳細な内容は[価格設定ドキュメント](/zh-CN/billing/pricing)で確認できます。

#### ワンストップ統合

Zeaburは強力なワンストップ統合機能を提供します。

あなたは一つのプロジェクト内で、フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージなど様々なサービスを同時にデプロイすることができます。これによって、異なるプラットフォームに個別にデプロイする必要がなくなります。
