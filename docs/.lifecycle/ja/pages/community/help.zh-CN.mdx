# ヘルプを求める

Zeaburを使用中に問題が発生した場合は、[公式Discordサーバー](https://discord.gg/DrdGCvXEyY)でヘルプを求めることができます。

![server](/community/help/server.png)

## ヘルプの受け方

Discordサーバーには「問題求助」というチャンネルがあり、そこで質問を投稿できます。

![help channel](/community/help/help-channel.png)

### 投稿の形式

問題を説明する際には、以下の形式に従うことで、問題を素早く特定し解決できるかもしれません。

- 投稿のタイトルに簡潔な問題の説明を記述します
- コンテンツ部分で問題の詳細を記述します
- 投稿にあなたの「projectID」、「serviceID」または「deploymentID」を提供してください。これらはZeaburコンソールのURLから見つけることができます。
  ![url](/community/help/url.png)
