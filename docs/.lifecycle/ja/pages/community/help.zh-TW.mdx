# ヘルプを求める

時々、Zeaburを使用中に問題が発生することがあります。その際は[公式Discordサーバー](https://discord.gg/DrdGCvXEyY)でヘルプを求めることができます。

![server](/community/help/server.png)

## ヘルプの手順

当社のDiscordサーバーには、「ヘルプを求める」というチャンネルがあります。こちらで問題を投稿できます。

![help channel](/community/help/help-channel.png)

### 投稿のフォーマット

問題を記述する際に以下のフォーマットに従うと、問題の特定と解決が迅速に進むでしょう。

- 投稿のタイトルに問題を簡潔に記述
- コンテンツエリアに問題の詳細を記述
- 投稿にあなたの`projectID`、`serviceID`、または`deploymentID`を提供してください。これらはZeaburコンソールのURLから取得できます。
  ![url](/community/help/url.png)
