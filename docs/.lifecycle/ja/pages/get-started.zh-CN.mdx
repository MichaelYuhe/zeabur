# クイックスタート

## ステップ1：アカウントの作成

Zeaburへのサービスのデプロイを開始するには、まずアカウントを作成する必要があります。
現在ZeaburはGitHubログインのみをサポートしています。GitHubアカウントを持っていない場合は、[ここ](https://github.com/join)で作成できます。

[Zeaburのログインページ](https://dash.zeabur.com/login)にアクセスすると、GitHubでログインするボタンが表示されます。
そのボタンをクリックすると、GitHubの認証ページにリダイレクトされます。「Authorize」ボタンをクリックしてZeaburにGitHubアカウントへのアクセスを許可してください。

![sign in](/start/sign-in.png)

## ステップ2：プロジェクトの作成

Zeaburでは、プロジェクトは実際の工学プロジェクトを表しています。一つのプロジェクトで、実際のニーズに合わせて複数の異なるタイプのサービスをデプロイできます。

認証が成功すると、自動的にZeaburコントロールパネルページにリダイレクトされます。
**新しいプロジェクトを作成** ボタンをクリックして新しいプロジェクトを作成します。または `CMD + k` のショートカットキーを押してクイックアクションメニューを開き、`新しいプロジェクトを作成` を選択することもできます。

プロジェクトの地理的な所在地を選ぶためのポップアップが表示されます。Zeaburは現在、`asia-east` にあるGCPと `us-west` にあるAWSの2つのリージョンをサポートしています。
サービスをデプロイしたいリージョンを選択し、我々はあなたのためにランダムな名前でプロジェクトを作成します。後でプロジェクト設定で名前を変更することができます。

![new project](/start/create-project.png)

<Callout>
  初めてプロジェクトを作成する場合、サービス条項を読んで同意する必要があるかもしれませんので、それに従ってください。
</Callout>

## ステップ3：サービスのデプロイ

作成したばかりのプロジェクトをクリックすると、Zeaburはそのプロジェクトの詳細ページを表示します。

![project](/start/project.png)

サービスをデプロイするには、**サービスを追加** ボタンをクリックします。すると、サービスタイプを選択するためのポップアップが表示されます。

![select service type](/start/select-service-type.png)

Zeaburは **Git Service** と **Marketplace Service** の2種類のサービスをサポートしています。

- **Git Service** では、ソースコードのデプロイが可能です。Zeaburは自動的にソースコードのビルドとデプロイを行います。
  初めて使用する場合は、ZeaburにGitHubのリポジトリへのアクセス権を設定する必要があります。

- **Marketplace Services** では、よく使われるサービスのマーケットプレイスからサービス（データベースやオープンソースツールなど）をデプロイできます。私たちのオープンソースのサービスマーケットプレイスリストは[ここ](https://github.com/zeabur/zeabur/tree/main/marketplace)で見ることができます。

デプロイしたいサービスのタイプを選択後、あとはデプロイしたいサービスを自由に探索できます。
